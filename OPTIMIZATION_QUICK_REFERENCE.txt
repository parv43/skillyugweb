⚙️  HOW TO GENERATE BLOG THUMBNAILS (Phase 3)
═══════════════════════════════════════════════════════════════════════════

Option 1: Using Sharp CLI (Recommended)
$ npm install -g sharp-cli
$ for img in unsplash-images/*.jpg; do
    npx sharp -i "$img" -resize 800 600 \
      -o "public/blog-thumbnails/$(basename $img .jpg).webp"
  done

Option 2: Using ImageMagick
$ convert unsplash-image.jpg -resize 800x600 -quality 75 \
    public/blog-thumbnails/best-ai-tools.webp

Option 3: Online Tool (simplest for few images)
1. Go to: https://webp.online-convert.com/
2. Upload JPG, set 800x600, quality 75
3. Download WebP, save to public/blog-thumbnails/

Expected results:
- Each thumbnail: 15-30 KB
- 20 articles: 300-600 KB total
- vs Original: 10-16 MB (20x improvement!)
